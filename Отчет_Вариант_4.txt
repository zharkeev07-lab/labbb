ФОРМУЛИРОВКА ЗАДАНИЯ:
─────────────────────
Пусть x₁=x₂=x₃=1; xᵢ=xᵢ₋₁+xᵢ₋₃; i=4,5,… Найти ∑(i=1 до 20) xᵢ/2ⁱ


КОД РЕШЕНИЯ:
─────────────
# Инициализация массива для хранения 20 чисел
x = [0] * 21  # Индексация с 1 до 20, поэтому размер 21
x[1] = x[2] = x[3] = 1

# Заполняем массив по формуле x_i = x_{i-1} + x_{i-3}
for i in range(4, 21):
    x[i] = x[i-1] + x[i-3]

# Вычисляем сумму
total_sum = 0
for i in range(1, 21):
    total_sum += x[i] / (2**i)

print(f"Результат суммы: {total_sum}")


ОБЪЯСНЕНИЕ КОДА:
─────────────────
1. x = [0] * 21
   - Создаем список из 21 нуля (индексы 0-20)
   - Нам нужно 21 место, чтобы удобно обращаться к элементам с индексами 1-20
   - Индекс 0 останется неиспользованным, это логично для нумерации

2. x[1] = x[2] = x[3] = 1
   - Устанавливаем начальные значения последовательности
   - Все первые три члена равны 1 по условию

3. for i in range(4, 21): x[i] = x[i-1] + x[i-3]
   - Это основной алгоритм (рекуррентная формула)
   - Каждый новый элемент равен сумме элемента на 1 позицию назад и на 3 позиции назад
   - Цикл начинается с 4, так как первые три элемента уже инициализированы
   - Заполняем массив: 1, 1, 1, 2, 3, 4, 6, 9, 13, 19, 28, 41, 60, 88, ...

4. total_sum += x[i] / (2**i)
   - Суммируем дроби, где числитель - элемент последовательности
   - Знаменатель - степень двойки (2^i)
   - 2**i эквивалентно 2^i

ФОРМУЛИРОВКА ЗАДАНИЯ:
─────────────────────
Дан массив случайных чисел n элементов. Поменять значения переменных A, B, C. 
Например, до обмена: A=1, B=2, C=3, после процедуры обмена: A=3, B=1, C=2.


КОД РЕШЕНИЯ:
─────────────
import random

n = 10
arr = [random.randint(1, 100) for _ in range(n)]
print(f"Исходный массив: {arr}")

# Разворачиваем массив
arr.reverse() 

print(f"Развернутый массив: {arr}")


ОБЪЯСНЕНИЕ КОДА:
─────────────────
1. import random
   - Импортируем модуль для генерации случайных чисел

2. n = 10
   - Определяем размер массива (10 элементов)

3. arr = [random.randint(1, 100) for _ in range(n)]
   - Генератор списков (list comprehension)
   - Создает массив из n случайных целых чисел
   - randint(1, 100) генерирует целые числа от 1 до 100 включительно
   - for _ in range(n) - цикл, повторяющийся n раз
   - Подчеркивание (_) используется, так как переменная цикла нам не нужна

4. arr.reverse()
   - Встроенный метод Python, который переворачивает список на месте
   - Первый элемент становится последним, последний - первым
   - После вызова исходный список изменяется
   - Это эквивалентно arr = arr[::-1], но более эффективно

ЗАДАЧА 4: РАБОТА С МАТРИЦЕЙ n×n
═══════════════════════════════════════════════════════════════════════════════

ФОРМУЛИРОВКА ЗАДАНИЯ:
─────────────────────
Дан файл, компоненты которого являются целыми числами. Записать во второй файл
числа, являющиеся точными квадратами четных чисел.


КОД РЕШЕНИЯ:
─────────────
import random

n = 5
k = 2  # Например, строка 2 (индексация с 0)
matrix = [[random.randint(1, 50) for _ in range(n)] for _ in range(n)]

# 1. Обнуление строк выше k
for i in range(k):
    for j in range(n):
        matrix[i][j] = 0

# 2. Сбор элементов ниже строки k, сортировка и возврат
below_elements = []
for i in range(k + 1, n):
    below_elements.extend(matrix[i])

below_elements.sort(reverse=True)

# Заполняем обратно
idx = 0
for i in range(k + 1, n):
    for j in range(n):
        matrix[i][j] = below_elements[idx]
        idx += 1


ОБЪЯСНЕНИЕ КОДА:
─────────────────
1. matrix = [[random.randint(1, 50) for _ in range(n)] for _ in range(n)]
   - Вложенные списки (двумерный массив/матрица)
   - Внешний список содержит n строк
   - Каждая строка - это список из n элементов
   - Заполняем случайными числами от 1 до 50

2. Обнуление строк выше k:
   - Используем двойной цикл (по строкам i и столбцам j)
   - Условие i < k означает строки выше k-й строки
   - Устанавливаем matrix[i][j] = 0 для всех таких элементов

3. below_elements = []
   - Создаем пустой список для хранения элементов ниже k-й строки

4. below_elements.extend(matrix[i])
   - Метод extend() добавляет все элементы строки в список
   - Это эквивалентно нескольким вызовам append()
   - Собираем все элементы из строк k+1, k+2, ..., n-1 в один список

5. below_elements.sort(reverse=True)
   - Сортируем собранные элементы в порядке убывания
   - reverse=True означает от большего к меньшему

6. Заполняем обратно:
   - Проходим по тем же строкам (k+1 и ниже)
   - Заполняем отсортированными элементами в порядке возрастания индекса

ЗАДАЧА 5: СРЕДНЕЕ ГЕОМЕТРИЧЕСКОЕ И ЗАМЕНА
═══════════════════════════════════════════════════════════════════════════════

ФОРМУЛИРОВКА ЗАДАНИЯ:
─────────────────────
Дано число n, матрица 4×4. Найти среднее геометрическое главной диагонали и 
заменить им максимальный элемент побочной диагонали. Положительные элементы 
матрицы обнулить, остальные не менять.


КОД РЕШЕНИЯ:
─────────────
import math

matrix = [[random.randint(-10, 10) for _ in range(4)] for _ in range(4)]

# 1. Главная диагональ (элементы [0][0], [1][1], [2][2], [3][3])
prod = 1
for i in range(4):
    prod *= matrix[i][i]
geom_mean = prod**(1/4)

# 2. Поиск макс. элемента побочной диагонали ([0][3], [1][2], [2][1], [3][0])
side_diag = [matrix[i][3-i] for i in range(4)]
max_val = max(side_diag)

# Замена
for i in range(4):
    if matrix[i][3-i] == max_val:
        matrix[i][3-i] = geom_mean
        break

# 3. Обнуление положительных элементов
for i in range(4):
    for j in range(4):
        if matrix[i][j] > 0:
            matrix[i][j] = 0


ОБЪЯСНЕНИЕ КОДА:
─────────────────
1. matrix = [[random.randint(-10, 10) for _ in range(4)] for _ in range(4)]
   - Матрица 4×4 со случайными целыми числами от -10 до 10

2. Вычисление среднего геометрического:
   - prod = 1 (инициализируем произведение)
   - Главная диагональ: [0][0], [1][1], [2][2], [3][3]
   - prod *= matrix[i][i] умножает prod на элемент диагонали
   - geom_mean = prod**(1/4) вычисляет корень 4-й степени из произведения
   - Формула: ⁴√(a·b·c·d)

3. side_diag = [matrix[i][3-i] for i in range(4)]
   - Побочная диагональ: [0][3], [1][2], [2][1], [3][0]
   - Индекс столбца = 3 - индекс строки
   - Генератор списков собирает все элементы побочной диагонали

4. max_val = max(side_diag)
   - Находим максимальный элемент побочной диагонали

5. Замена:
   - Проходим по побочной диагонали
   - Находим первый элемент, равный максимуму
   - Заменяем его на среднее геометрическое
   - break останавливает цикл после первой замены

6. Обнуление положительных элементов:
   - Двойной цикл по всей матрице
   - Если элемент > 0, устанавливаем его в 0
   - Отрицательные и нулевые элементы остаются без изменений
